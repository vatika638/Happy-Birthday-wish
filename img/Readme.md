# Adding Photos and Videos

To personalize this template, add your own photos and videos to this folder.

## Steps

1. Place your **photos or videos** inside this folder.
2. Rename your photos to match the filenames used in the `index.html` file.

Example format:
img/name_of_your_photo.jpg
img/name_of_your_photo.png
img/name_of_your_photo.jpeg

Make sure the filename in `index.html` matches the name of your image file exactly.
